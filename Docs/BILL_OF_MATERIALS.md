# Bill of Materials

## Core
- 1 x STM32 NUCLEO-L476RG
- 1 x breadboard
- jumper wires
- USB cable

## Inputs
- 1 x LM35 or TMP36 analog temperature sensor
- Optional I2C EEPROM / OLED module
- Optional SPI memory / peripheral module
- 1 x MQ-2 or MQ-135 gas sensor module
- 1 x SW-420 vibration sensor module or dry-contact vibration switch
- 1 x float switch / pressure switch / limit switch

## Outputs
- 1 x green LED + 220 ohm resistor
- 1 x red LED + 220 ohm resistor
- 1 x active buzzer
- 1 x NPN transistor for buzzer driver
- 1 x relay module or NPN + relay + flyback diode

## Optional
- 1 x RS-485 module
- 1 x 16x2 LCD or OLED
- 1 x microSD logging module
